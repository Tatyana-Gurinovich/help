git clone ссылка на репозиторий - первое действие
git pull - скачивание файлов из online репозитория на локальную машину (git add .- если добовляем файлы в пустой репозиторий)
git status - показывает состояние репозитория
git commit -m " " - сохроняет изменения в репозитории
git push - закидывает всё на online репозиторий(github, gitlab etc.)
